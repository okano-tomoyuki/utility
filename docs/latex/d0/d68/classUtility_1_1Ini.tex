\doxysubsection{Detailed Description}
Iniファイルパース用クラス 

\begin{DoxyNote}{Note}
本クラス作成に当たり以下のサイトを参考にした。 ~\newline
 クロスプラットフォームのini ファイルパーサ ~\newline
 \mbox{\hyperlink{}{https\+://github.\+com/\+Rookfighter/inifile-\/cpp.\+git}} ~\newline
 本家と異なる点は以下のとおり。 ~\newline
 読込時に全セクション・全フィールドを読み込まず、read$\sim$ やwrite$\sim$ でそれぞれ指定したものだけ読み取れるようにする。 ~\newline
 上記に伴い対象のセクション・フィールドがiniファイル中に存在しなかった場合第3引数で指定したデフォルト値を読み取る。 
\end{DoxyNote}
